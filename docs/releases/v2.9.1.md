## Changes

### 1. MAB Context-Aware Optimization

**Implementation:**
- Added `_cosine_similarity()` - cosine similarity for feature vectors
- Added `find_similar_contexts()` - find top-K similar contexts
- Added `predict_with_similarity()` - similarity-weighted voting
- Activates when `_context_vectors > 5`

**New Tests:** 21 new tests (56 total), 92% coverage
